Project Description
This project implements a preprocessing pipeline for unstructured documents to prepare them for Large Language Model (LLM) applications.

The system supports multiple file formats including PDF, Word, Excel, PowerPoint, and EPUB. It extracts text from the selected document, cleans and normalizes the content, splits it into 500-word chunks

How to Run the Code
1. Install Required Libraries
Run the following command to install the required dependencies:

pip install PyPDF2
pip install PyPDF2 python-docx openpyxl nltk

2. Run the cells
  
3. Select a File
